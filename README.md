## 📊 Analyst Notes on Methodology

> A few principles I apply consistently across analyses:

- **Assumption checking first** — Chi-square expected cell counts, regression residuals, and collinearity are always verified before reporting
- **Honest R²** — A weak model is reported as weak, with explanation of what factors are likely missing
- **Effect size alongside p-values** — Statistical significance alone doesn't determine importance
- **Multiple-response handling** — Properly converted to binary dummies before frequency analysis, not treated as single-choice
- **Ceiling/floor effects** — Flagged when present, as they affect correlation and regression power
