// 03 — Approach

Analyst Principles

⚠️

Assumption Checking First

Expected cell counts, residuals, and collinearity verified before any result is reported.

📉

Honest R²

A weak model is reported as weak — with clear explanation of missing variance drivers.

📏

Effect Size + p-values

Statistical significance alone does not determine importance. Both always reported together.

🔢

Multiple-Response Handling

Items converted to binary dummies before frequency analysis, never treated as single-choice.

📊

Ceiling / Floor Effects

Flagged when present — they reduce correlation and regression power.

🗂️

Reproducible Outputs

Each project documented with full methods, figures, and outputs.
